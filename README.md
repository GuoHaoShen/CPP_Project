CPP_Project
------------
本项目为个人c++项目仓库，包含学习过程中各种项目代码，持续更新中...

Environment
------------
Ubuntu20.04

Make\Install
------------
进入对应子模块后执行：
```
cd matplotlib-cpp-master
mkdir build && cd build
cmake ..
make
```

List
----------
* matplotlib_test
  
  matplotlibcpp的使用案例，需要先安装matplotlib-cpp

* matplotlib-cpp

    matplotlib-cpp的源码，安装方法见模块中的README.md

* numerical_optimization

    机器人中的数值优化源码
